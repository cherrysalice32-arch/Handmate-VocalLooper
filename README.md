# Handmate VocalLooper

## Overview
Handmate VocalLooper is a powerful tool designed for vocalists, podcasters, and performers to create layered vocal loops, apply effects, and enhance live performances.

## Features
- **Multi-Track Looping:** Record and layer multiple vocal tracks seamlessly.
- **Real-Time Effects:** Apply effects like reverb, delay, and pitch correction on-the-fly.
- **User-Friendly Interface:** Intuitive interface allows for easy navigation and control.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/cherrysalice32-arch/Handmate-VocalLooper.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Handmate-VocalLooper
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage
To start the application, run:
```bash
npm start
```
